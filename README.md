<h1 align="center">🎵 Bem-vindo(a) ao Projeto <b>Eko Music</b> 🎶</h1>
<p align="center">
Leve, moderno e intuitivo — um streaming musical usando <b>HTML, CSS e JavaScript</b>.
</p>

---

<h2>🚀 Sobre o Projeto</h2>
<p>
O <b>Eko Music</b> tem como objetivo oferecer uma <b>plataforma leve e acessível</b> para reprodução de músicas online.  
Ele simula um serviço completo de streaming musical com autenticação, catálogo, player e biblioteca pessoal.
</p>

<ul>
<li>🧠 Ideal para estudos e fins educacionais</li>
<li>🎧 Protótipos e demonstrações de sistemas de streaming</li>
<li>⚙️ Base sólida para projetos mais robustos</li>
</ul>
