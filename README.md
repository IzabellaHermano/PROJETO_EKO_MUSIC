<h2><i class="fa-solid fa-rocket"></i> Sobre o Projeto</h2>
<p>
O <b>Eko Music</b> tem como objetivo oferecer uma <b>plataforma leve e acessível</b> para reprodução de músicas online.  
Ele simula um serviço completo de streaming musical com autenticação, catálogo, player e biblioteca pessoal.
</p>

<ul>
  <li><i class="fa-solid fa-book-open"></i> Desenvolvido para estudos e fins educacionais</li>
  <li><i class="fa-solid fa-compact-disc"></i> Protótipos e demonstrações de sistemas de streaming</li>
  <li><i class="fa-solid fa-gears"></i> Base sólida para projetos mais robustos</li>
</ul>
